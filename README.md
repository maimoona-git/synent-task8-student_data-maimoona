 Student Performance Prediction Project

 Objective

The objective of this project is to build a Machine Learning model to predict student performance based on academic and personal factors.

Dataset

Dataset Used: Student Performance Dataset

Format: CSV File

Source: Kaggle
 Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Google Colab

 Tasks Performed

* Imported CSV dataset
* Checked missing values
* Performed data preprocessing
* Converted categorical data
* Selected important features
* Split dataset into training and testing data
* Trained Machine Learning model
* Evaluated model accuracy

 Machine Learning Models Used

* Linear Regression
* Decision Tree
* Random Forest

 Evaluation Metrics

* Accuracy Score
* RMSE (Root Mean Square Error)

 Key Insights

* Study time positively affects student performance.
* Attendance and previous scores influence final results.
* Random Forest model provided better prediction accuracy.
Output

A Machine Learning model capable of predicting student performance with good accuracy.
